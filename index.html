🧠 What in the World is DNS?

DNS = Domain Name System, aka the phonebook of the internet 📞🌐.

Your computer’s like: “Bro, I speak IP, not human 😩. What even is this ‘google dot com’??”

So it calls up DNS like a chaddi buddy and goes, “Hey fam, what’s Google’s address?”

🛣️ The DNS Journey (aka the “Find My IP” quest):

  1. Local Cache: “Ah, I remember that IP 😎!” ✅ Done. No drama.
  2. DNS Resolver: ISP’s internet detective 🕵️‍♂️.
  3. Root Server: Points to `.com` directory like grandma 👵.
  4. TLD Server: “Google lives with this authoritative server 👇.”
  5. Authoritative Server: “That’s 142.250.77.206 my dear!”
  6. DNS Resolver returns like a hero 🦸 → browser says: “Goooooogle, here I comeeeee!”

🌳 DNS Structure – Not as Boring as It Sounds!

  • DNS Records: IPs stored like a family tree 👨‍👩‍👧‍👦.
  • Namespace: Names sorted like school roll call 📋.
  • Name Server: Mini-database saying name = IP.

🌐 Types of Domains

  • Generic: `.com`, `.org`, `.net` — cool uncles/aunties.
  • Country: `.in`, `.us` — desi versions 🇮🇳🇺🇸.
  • Inverse Domain: Reverse lookup 🔍.

🧩 DNS Resolver – The Wingman

A DNS resolver starts the lookup. It’s like:

“Don’t worry fam, I got this 🕶️.”

🔄 Types of DNS Queries

  • Recursive: "Get me the answer or admit defeat!" 😤
  • Iterative: "Tell me what you know, I’ll ask around." 🤝
  • Non-Recursive: “Already cached, bro. Here you go!” 😎

🧠 DNS Caching

Resolver stores past addresses like an elephant 🐘 — *almost* never forgets.

📛 Domain Hierarchy – Who’s Who?

  • TLD: `.com`, `.org`, `.biz`, `.in` — internet surname 👑
  • SLD: Part before the dot → “geeksforgeeks” in `geeksforgeeks.org`
  • Subdomains: Babies like `admin.geeksforgeeks.org`, stackable 🍔

📂 DNS Record Types

  • A Record: Maps to IPv4 📍
  • AAAA Record: Maps to IPv6 💪
  • CNAME Record: Nicknames 🎭
  • MX Record: Email routing 📬
  • TXT Record: Texty configs 😤
